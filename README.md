# 🪖 AI-IoT Smart Helmet for Two-Wheeler Safety

## 📌 Overview
The AI-IoT Smart Helmet is an intelligent safety system designed to improve the protection of two-wheeler riders. It combines Artificial Intelligence, IoT sensors, and embedded systems to monitor rider safety, detect accidents, and send emergency alerts with the rider's live location.

This project aims to reduce road accident fatalities by enabling real-time monitoring and rapid emergency response.

---

## ✨ Features

- 🪖 Helmet wear detection
- 🚫 Ignition lock when the helmet is not worn
- 😴 Drowsiness detection
- 💥 Accident detection
- 📍 GPS-based live location tracking
- 📲 Emergency alert notifications
- 🌫️ Gas and smoke detection
- 📡 IoT-based real-time monitoring

## ⚙️ Working

1. Detects whether the rider is wearing the helmet.
2. Prevents vehicle ignition if the helmet is not worn.
3. Continuously monitors rider movement.
4. Detects accidents using motion sensors.
5. Retrieves the rider's GPS location.
6. Sends emergency alerts to predefined contacts.
7. Monitors hazardous gas levels for rider safety.
